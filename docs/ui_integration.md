# 界面集成说明

当前界面已连接基础仿真核心，用户可以通过主窗口输入参数、启动仿真、暂停/继续后台任务、重置状态，并查看关键结果。

## 界面状态

- 参数区负责表达一维 Stefan 问题的基础几何、温度、物性和时间设置。
- 状态区负责展示当前时间、相变界面位置、运行进度、状态和提示信息。
- 结果区负责通过选项卡展示温度分布图、相界面-时间图和温度云图。
- 顶部控制区提供参数加载、参数保存和仿真结果导出入口。

## 线程边界

- UI 主线程只负责收集参数、启动任务、接收信号和刷新界面。
- 仿真计算运行在后台 worker 中，完成后返回 `SimulationResult`。
- 错误信息通过统一异常边界转为界面提示，不直接中断主窗口。

## 后续扩展

- 可继续增强绘图交互、结果导出体验和更细粒度的过程动画。
- 可在当前后台任务边界内接入更高精度求解器。
