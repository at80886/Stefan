# 结果导出与参数文件说明

当前应用支持将仿真参数保存为 JSON 文件，并将一次仿真结果导出为目录。

## 参数文件

- 参数文件使用 UTF-8 JSON 格式。
- 默认示例位于 `resources/examples/default_case.json`。
- 应用关闭时会把当前参数记录到用户本地应用数据目录。
- 应用启动时会优先读取上次退出时记录的参数。
- 如果没有上次参数记录，则读取默认示例参数填充参数区。
- 用户可以在界面中加载参数文件，也可以保存当前参数。

## 结果导出

导出结果以目录形式保存，包含以下文件：

- `parameters.json`：本次仿真使用的参数。
- `summary.csv`：全部参数设置和最终相界面位置。
- `interface.csv`：时间与相界面位置历史。
- `temperature.csv`：最终时刻的空间温度分布。

## 后续扩展

- 后续可增加图像导出、导出压缩包和更严格的数值验证报告。
