# 实时显示更新说明

仿真运行时，后台 worker 会通过进度信号传回当前输出帧。每个输出帧包含当前时间、相界面位置、空间坐标和当前温度分布。

## 数据流

- 求解器在每个输出步生成 `SimulationState`。
- `SimulationState` 携带单帧温度快照，用于运行中的图像更新。
- 主窗口累积这些状态，临时组成 `SimulationResult`。
- 结果显示区继续统一消费 `SimulationResult`，避免为实时显示重复定义绘图入口。

## 界面行为

- 运行时同步更新状态面板、温度分布、界面位置和云图。
- 暂停时保留当前图像和状态信息。
- 继续后沿用已有历史继续追加显示。
- 重置或加载新参数时清空运行中的临时结果。

## 刷新控制

- worker 对进度信号做轻量节流，降低高频输出对主线程的影响。
- 主窗口对绘图刷新做轻量节流，避免每个进度快照都触发重绘。
- 最终结果到达后仍使用完整 `SimulationResult` 覆盖运行中的临时结果。
