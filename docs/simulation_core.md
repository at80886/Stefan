# 仿真核心说明

当前仿真核心提供一维 Stefan 问题的基础数据与求解能力，保持与 PyQt6 界面解耦。

## 数据对象

- `StefanParameters` 表达几何、温度边界、物性近似参数、时间步长和输出间隔。
- `SimulationState` 表达仿真过程中的时间、总时长、界面位置、进度、状态和单帧温度快照。
- `SimulationResult` 表达空间坐标、时间序列、温度历史、界面轨迹和结果摘要。
- `StefanAppError` 表达可记录、可提示、可恢复的应用错误。

## 当前能力

- 基线求解器能够生成温度分布历史、相变界面位置历史和实时显示所需的输出帧。
- 求解器不依赖 PyQt6，可在后台任务、测试和导出流程中复用。
- 参数错误或显式格式稳定性问题会通过统一异常类型报告。

## 后续扩展

- 后续可在现有接口下替换或增加更高精度的 Stefan 问题求解器。
- UI、绘图和导出模块应优先消费 `SimulationResult`，避免重复定义结果结构。
