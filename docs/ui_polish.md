# UI 美化说明

当前界面采用暗色调工业仪表盘风格，用于突出仿真图像、关键数值和运行状态。

## 视觉结构

- 主窗口使用深色背景，顶部控制区、参数区、结果区和状态区以低对比面板分隔。
- 按钮、输入框、选项卡和状态栏由 `resources/styles/app.qss` 统一管理。
- 状态区的时间、界面位置和进度使用更醒目的数值样式。
- 运行、暂停、完成和错误状态通过不同颜色的状态标签区分。

## 图表风格

- 温度分布图、界面位置图和云图使用暗色画布与高对比坐标信息。
- 温度曲线使用青绿色强调，界面轨迹使用琥珀色强调。
- 云图保留冷暖色温度映射，并在暗色主题下强化相界面标注。

## 维护要求

- 后续 UI 调整应优先复用现有对象名和 QSS 样式。
- 绘图颜色应继续保证暗色背景下的坐标、刻度、图例和云图标注可读。
- UI 美化不应改变仿真核心、后台任务、结果数据模型和导出模块的职责边界。
