# 03_UI_INTEGRATION：建立最小可交互仿真界面

## 任务内容

- 将一维 Stefan 问题仿真核心接入 PyQt6 主界面。
- 提供用户可编辑的基础仿真参数输入区域。
- 提供启动、暂停、重置和重新运行仿真的界面控制入口。
- 在界面中展示当前仿真时间、相变界面位置、运行状态和错误提示。
- 将仿真运行过程与 UI 主线程职责分离，避免界面在计算期间失去响应。
- 为温度分布和相变界面轨迹预留可视化显示区域。
- 让默认参数能够支持用户打开应用后完成一次最小仿真操作。

## 目标效果

- 应用从静态窗口进入可操作状态，用户能够在界面中发起基础仿真。
- 用户能够看到仿真状态和关键结果随运行过程更新。
- 仿真错误能够以可理解的方式反馈到界面，并保留可恢复空间。
- UI、仿真核心和结果展示之间形成清晰的数据流。
- 后续任务可以继续围绕绘图、导出和交互细节扩展。
