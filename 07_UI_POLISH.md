# 07_UI_POLISH：美化 UI 界面与工业仪表盘风格

## 任务内容

- 美化现有 PyQt6 主界面，使整体视觉从基础工具界面提升为暗色调工业仪表盘风格。
- 统一主窗口、顶部控制区、参数区、状态区和结果显示区的视觉层级。
- 强化仿真运行状态的可读性，让空闲、运行、暂停、完成和错误状态更容易区分。
- 优化参数输入控件、按钮、选项卡和状态文本的观感，使界面更适合长时间观察和重复操作。
- 让温度分布图、相界面轨迹图和云图在暗色主题下保持清晰的坐标、刻度、图例和颜色对比。
- 为关键数值输出提供更醒目的仪表盘式呈现，突出当前时间、界面位置和运行进度。
- 保持示例参数、参数保存加载、结果导出和实时显示流程在新界面风格下自然可用。
- 保持 UI 美化不改变现有仿真核心、后台任务、结果数据模型和导出职责边界。

## 目标效果

- 用户打开应用后能直接感受到稳定、专业、偏工业监测场景的主界面风格。
- 暗色界面能够突出仿真图像、关键数值和运行状态，减少无关视觉干扰。
- 参数配置、仿真控制、过程观察和结果导出之间的操作关系更加清楚。
- 实时显示过程中，图表变化、状态提示和数值面板形成一致的仪表盘体验。
- 后续任务可以继续围绕交互读数、图像保存、动画效果和发布打包扩展。
