# 02_SIMULATION_CORE：建立一维 Stefan 问题仿真核心

## 任务内容

- 建立一维 Stefan 问题的核心仿真能力。
- 完善仿真参数、边界条件、运行状态和结果数据的表达。
- 让应用能够根据用户给定参数生成温度分布历史。
- 让应用能够生成相变界面位置随时间变化的历史数据。
- 提供仿真运行状态、错误信息和结果摘要所需的数据支撑。
- 保持仿真核心与 PyQt6 界面解耦，便于后续放入后台任务中运行。
- 为后续绘图展示、结果导出和 UI 控件联动提供稳定数据来源。

## 目标效果

- 项目具备一个可被界面调用的基础仿真核心。
- 仿真参数、结果和运行状态的含义清晰，后续模块可以直接复用。
- 基线仿真能够产出温度曲线数据和相变界面轨迹数据。
- 异常情况能够形成可记录、可提示、可恢复的错误信息。
- 后续开发可以围绕仿真核心继续扩展绘图、导出和交互控制。
