# 08_WINDOWS_RELEASE：发布 Win10/11 x64 应用程序

## 任务内容

- 为当前 PyQt6 应用建立面向 Windows 10/11 x64 的发布流程。
- 生成可直接启动的 Windows 桌面程序发布目录。
- 确保发布包包含界面样式、示例参数和应用运行所需资源。
- 保持发布流程与源码、测试和文档资源边界清晰。
- 使用项目指定的 conda `py39` 环境完成构建与验证。
- 让发布流程能够重复执行，避免依赖手工复制资源。
- 补充 Windows 发布流程相关说明文档。

## 目标效果

- 用户能够在发布目录中启动一维 Stefan 问题模拟器。
- 发布版界面保持中文、暗色工业仪表盘风格和现有交互能力。
- 示例参数、参数保存加载、仿真运行、实时显示和结果导出在发布版中可用。
- 后续版本可以沿用同一流程重新生成 Win10/11 x64 发布包。
